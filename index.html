🚀 Мульти-кнопка с соцсетями

Красивая и функциональная кнопка для вашего сайта

📱 Демонстрация

Нажмите кнопку "⚙️ Настройки" в левом верхнем углу, чтобы настроить ссылки на соцсети.

Затем нажмите на фиолетовую кнопку в правом нижнем углу, чтобы открыть меню соцсетей.

🎯 Как это работает:

1. Настройка: Укажите ссылки на ваши социальные сети в панели настроек

2. Применение: Нажмите "Применить" для обновления кнопок

3. Использование: Кликните на главную кнопку для открытия меню соцсетей

4. Копирование: Скопируйте настройки для использования на вашем сайте

✨ Особенности:

• Поддержка Telegram, WhatsApp, VK, Instagram, телефон

• Настраиваемые иконки для каждой соцсети

• Красивый дизайн с анимациями

• Адаптивность для всех устройств

🎨 Красивый дизайн

Современный градиентный дизайн с плавными анимациями и эффектами наведения

⚙️ Простая настройка

Удобная панель настроек для быстрого изменения ссылок и иконок

📱 Адаптивность

Отлично выглядит на всех устройствах - от мобильных до десктопов
