# 🚀 Мульти-кнопка с соцсетями

Красивая и функциональная кнопка для вашего сайта с настраиваемыми ссылками на социальные сети и Jivo чатом.

## ✨ Особенности

- 🎨 **Красивый дизайн** - современный градиентный дизайн с анимациями
- ⚙️ **Простая настройка** - удобная панель для изменения ссылок и иконок
- 📱 **Адаптивность** - отлично работает на всех устройствах
- 🚀 **Быстрая работа** - легкий код без зависимостей
- 🔧 **Гибкость** - поддержка 20+ популярных платформ
- 💬 **Jivo интеграция** - встроенный чат с автоматическим скрытием стандартной кнопки
- 🎯 **Готовые решения** - продвинутая версия для немедленного использования
- 📋 **Селектор соцсетей** - выбор из 20 популярных платформ при добавлении кнопки
- 🎨 **Библиотека иконок** - готовые ссылки на иконки популярных соцсетей
- 🎨 **Кастомизация** - настройка цвета, формы и иконки главной кнопки
- 📋 **Улучшенный UX** - автоматический скролл к готовому коду с выделением
- 🔄 **Режим чата** - главная кнопка может работать как кнопка Jivo чата

## 📖 Как использовать

### 1. Настройка
1. Откройте [multibutton.html](multibutton.html) в браузере
2. Нажмите кнопку "⚙️ Настройки" в левом верхнем углу
3. **Добавление кнопок:**
   - Нажмите "➕ Добавить новую кнопку"
   - Выберите соцсеть из селектора (20 популярных платформ)
   - Или нажмите "🎨 Создать собственную кнопку" для кастомной
4. Заполните ссылки на ваши социальные сети
5. Настройте Jivo чат (если нужно)
6. Нажмите "Применить"

### 2. Новый UX - Автоматический скролл к коду
После нажатия "Применить":
- ✅ Окно настроек **НЕ закрывается**
- ✅ Происходит **плавный скролл** к блоку с готовым кодом
- ✅ Блок с кодом **выделяется** зеленой рамкой и тенью
- ✅ Кнопка "Копировать готовый код" **подсвечивается** желтым
- ✅ Через 3 секунды все выделения **автоматически убираются**

### 3. Тестирование
1. Нажмите на фиолетовую кнопку в правом нижнем углу
2. Откроется меню с вашими соцсетями
3. Кликните на любую кнопку для перехода

### 4. Копирование готового кода
1. После применения настроек автоматически откроется блок с кодом
2. Нажмите "📋 Копировать готовый код"
3. Вставьте код на ваш сайт перед тегом `</body>`

## 🛠️ Установка на сайт

### Вариант 1: Использование готового кода (рекомендуется)
1. Настройте кнопку в [multibutton.html](multibutton.html)
2. Скопируйте готовый код из блока "Настройки применены"
3. Вставьте код на ваш сайт перед тегом `</body>`

### Вариант 2: Быстрый старт
Скопируйте код из [simple_button.html](simple_button.html) и замените ссылки:

```html
<div class="multibutton">
    <button class="main-button" onclick="toggleMenu()">
        <img src="https://cdn-icons-png.flaticon.com/512/1827/1827933.png" alt="Меню">
    </button>
    <div class="popup-menu" id="popupMenu">
        <a href="https://t.me/ВАШ_ЮЗЕРНЕЙМ" class="social-button" target="_blank" title="Telegram">
            <img src="https://cdn-icons-png.flaticon.com/512/2111/2111646.png" alt="Telegram">
        </a>
        <!-- Добавьте другие соцсети -->
    </div>
</div>
```

## 🎨 Кастомизация

### Настройка главной кнопки
- **Цвет:** градиент по умолчанию или кастомный цвет
- **Форма:** круглая по умолчанию или кастомные размеры
- **Иконка:** стандартная или своя ссылка на изображение

### Настройка отображения
- **Способ открытия:** по клику или всегда видимое меню
- **Ориентация:** вертикальное или горизонтальное расположение

### Jivo интеграция
- **Режим чата:** главная кнопка = кнопка чата
- **Кастомная иконка:** своя иконка для Jivo
- **Автоматическое скрытие:** стандартная кнопка Jivo скрывается
