# База данных компаний

Этот проект содержит Excel файл с данными компаний, включающий информацию о названии, сайте, телефоне и email.

## Структура данных

Excel файл `companies_data.xlsx` содержит два листа:

### Лист "Финансовые компании"
Содержит данные финансовых компаний (85 компаний) со следующими колонками:
- **Название** - название компании
- **Сайт** - веб-сайт компании
- **Телефон** - контактный телефон
- **Email** - email адрес (или примечания, если email не указан)

### Лист "Общие компании"
Содержит данные различных компаний (40 компаний) со следующими колонками:
- **Название компании** - название компании
- **Сайт** - веб-сайт компании
- **Телефон** - контактный телефон (автоматически собран с сайтов)
- **Электронная почта** - email адрес (автоматически собран с сайтов)

## Содержимое

Файл содержит данные 125 компаний:
- **85 финансовых компаний** - включая займы, кредиты, микрокредиты
- **40 общих компаний** - различные отрасли (банки, клиники, магазины, сервисы и т.д.)

## Как использовать

### Установка зависимостей
```bash
pip3 install -r requirements.txt
```

### Создание Excel файла
```bash
python3 create_companies_excel.py
```

### Сбор контактной информации с сайтов
```bash
python3 scrape_contacts.py
```

### Просмотр файла
Откройте файл `companies_data.xlsx` в любом приложении, поддерживающем формат Excel (Microsoft Excel, LibreOffice Calc, Google Sheets и т.д.).

## Особенности

- Два отдельных листа для разных типов компаний
- Автоматический сбор контактной информации с сайтов
- Автоматическая настройка ширины колонок
- Стилизованные заголовки с серым фоном
- Центрированное выравнивание заголовков
- Поддержка кириллицы
- Структурированная организация данных
- Обработка SSL-ошибок и проблем с подключением

## Файлы проекта

- `companies_data.xlsx` - основной Excel файл с данными (2 листа)
- `create_companies_excel.py` - скрипт для создания Excel файла
- `scrape_contacts.py` - скрипт для автоматического сбора контактов с сайтов
- `requirements.txt` - зависимости Python
- `README.md` - этот файл с описанием

## Технические детали

Скрипт `scrape_contacts.py` использует:
- **requests** - для HTTP-запросов
- **BeautifulSoup** - для парсинга HTML
- **openpyxl** - для работы с Excel файлами
- **Регулярные выражения** - для извлечения телефонов и email
- **Обработка ошибок** - для работы с недоступными сайтами
