# Техническое задание на разработку кастомных отчетов

## 1.1. Отчеты по диалогам операторов

### 1.1.1. Отчет по завершённым диалогам в разрезе по операторам:
- Количество диалогов по каждому оператору (Артем Филипов, Татьяна Титова и др.)
- Распределение по статусам диалогов (Входящий, Исходящий)
- Количество диалогов по типам (Диалог завершен)
- Дата и время последнего сообщения
- Дата завершения оператором
- Длительность диалога (в минутах)
- Оценка клиента (1-5 баллов)
- Источник обращения (канал связи)

### 1.1.2. Отчет по завершённым диалогам в разрезе по каналам связи:
- Мобильное приложение
- New ZT.ru
- Распределение количества диалогов по каналам
- Среднее время обработки по каналам
- Средняя оценка клиентов по каналам
- Процент завершенных диалогов по каналам

### 1.1.3. Отчет по временным показателям диалогов:
- Дата создания диалога
- Дата последнего сообщения
- Дата завершения оператором
- Дата отзыва оператора
- Длительность диалога (расчет между датами)
- Среднее время обработки диалогов

### 1.1.4. Отчет по оценкам качества обслуживания:
- Количество оценок от 1 до 5 баллов
- Средняя оценка по операторам
- Процент положительных оценок (4-5 баллов)
- Распределение оценок по каналам связи
- Корреляция оценки с длительностью диалога

### 1.1.5. Отчет по активности операторов по датам:
- Количество диалогов по дням (17.04.2025, 18.04.2025)
- Распределение активности по операторам
- Пиковые дни активности
- Сравнение активности между датами

### 1.1.6. Отчет-график по созданию диалогов по часам:
- Распределение создания диалогов в течение дня
- Время пиковой активности создания диалогов
- Сравнение активности по дням недели

### 1.1.7. Отчет-график по завершению диалогов по часам:
- Время завершения диалогов операторами
- Распределение завершений в течение рабочего дня
- Эффективность работы операторов по времени

### 1.1.8. Отчет-дашборд с показателями по диалогам:
- Общее количество диалогов
- Количество активных операторов
- Средняя длительность диалога
- Общий рейтинг качества обслуживания
- Распределение по каналам связи
- Статус обработки диалогов

### 1.1.9. Отчет-таблица эффективности операторов:
- Рейтинг операторов по количеству обработанных диалогов
- Средняя оценка клиентов по каждому оператору
- Средняя длительность обработки диалога по операторам
- Процент завершенных диалогов

### 1.1.10. Отчет-диаграмма распределения нагрузки:
- Круговая диаграмма распределения диалогов между операторами
- Столбчатая диаграмма количества диалогов по каналам связи
- Гистограмма распределения оценок клиентов
- Временная диаграмма активности по дням
